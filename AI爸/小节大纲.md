# 《AI爸》小节大纲

**节标记格式**：`###1.` 至 `###12.`
**对话格式**：`""`
**人称**：第一人称（我 = 陈远）

###1. 凌晨两点
**主事件**：凌晨两点，我在书房用AI写小说，收到房贷短信
**子事件**：听到女儿房间有动静，过去一看只是翻身(动作) -> 回到电脑前，AI生成了一篇新稿子(发现) -> 房贷提醒短信弹出——"您本月应还款8700元"(冲突) -> 我盯着那个数字看了很久(反应)
**情绪**：压抑→麻木
**读者新获知**：主角被裁了，用AI写小说赚钱
**钩子**：老婆在卧室喊了一声"还不睡？"，我说"快了"。她不知道我在写小说。
**伏笔/物件**：手机（第1现：房贷短信）
**动静**：静
**对话密度**：低
**目标字数**：800字

###2. 奶粉罐
**主事件**：早上发现奶粉快喝完了，计算生活成本
**子事件**：老婆给小满喂奶，晃了晃空罐子(发现) -> "奶粉快没了"(对话) -> 我说"下班去买"(对话) -> 出门前看到鞋柜上的信用卡账单(发现) -> 地铁上算了一笔账：房贷8700+车贷2300+奶粉268+尿不湿≈这个月缺口还有8000(反应)
**情绪**：焦虑→无力
**读者新获知**：具体的欠款数字，巨大的经济压力
**钩子**：老婆发微信"晚上想吃什么？"我没回——因为我不知道怎么告诉她这个月可能连菜钱都不够了
**伏笔/物件**：奶粉罐（第1现：268元/罐）
**动静**：静
**对话密度**：中
**目标字数**：900字

###3. AI写稿
**主事件**：我在电脑前给AI写提示词，完成当天的小说
**子事件**：给AI打了一串提示词(动作) -> AI生成了开头，我改了半个小时(发现) -> 发布到平台，等审核(动作) -> 刷新后台——阅读量:23，收益:0.36元(反应) -> 看到一条新评论："加油，写的挺好的。"（发现）
**情绪**：麻木→微暖
**读者新获知**：AI写一篇小说收益很低，有个读者每篇都评论
**钩子**：我点开那个读者的主页——"晴天妈妈"，关注0，粉丝0，像是一个专门注册来看小说的号
**伏笔/物件**：晴天妈妈（首次出现，第1条铺垫）
**动静**：静
**对话密度**：低
**目标字数**：850字

###5. 她的手机
**主事件**：晚上我路过客厅，看到老婆在刷手机，屏幕上是小说页面
**子事件**：我出来倒水，看到老婆靠在沙发上刷手机(动作) -> 她飞快地锁屏了(冲突) -> "看什么呢？"(对话) -> "没什么，刷朋友圈。"(对话) -> 但她锁屏之前我瞥到了——是一个小说阅读页面(发现)
