# 小节大纲.md — 《厝》

## 节1 | 接到电话
主事件：陈潮生在深圳加班，接到母亲电话说父亲摔了一跤
子事件：加班的日常 -> 电话进来，母亲声音不对 -> 「你爸摔了」 -> 挂电话后愣住
情绪：心慌→愧疚（强度5→7）
读者新获知：父亲已经62岁了，一个人住在老厝
钩子：开车回家的路上
动静：静
对话密度：中

## 节2 | 回到老厝
主事件：陈潮生深夜回到老厝，看到父亲坐在茶盘前
子事件：开车回家一路沉默 -> 推开老厝的门 -> 父亲手抖着泡茶 -> 看到父亲第一反应是「食未？」
情绪：心酸（强度7）
读者新获知：父亲老了，手在抖，但工夫茶的流程一个没少
钩子：茶具还是30年前那套
动静：静
对话密度：中
伏笔/物件：工夫茶具第1现

## 节3 | 回忆·拜老爷
主事件：回忆童年——父亲硬拉陈潮生去拜老爷
子事件：大年初一被拉去祠堂 -> 跪在蒲团上听不懂父亲念什么 -> 香灰掉在手臂上 -> 父亲粗糙的手把他的袖子拉下来
情绪：憋屈→隐约的被在乎（强度6）
读者新获知：父亲信老爷，会念保佑儿子的话
钩子：父亲的手
动静：动静
对话密度：低

## 节4 | 回忆·工夫茶的规矩
主事件：回忆童年——父亲从不让他碰茶具
子事件：偷摸茶壶被喝止 -> 父亲自己做茶，他只能在旁边看 -> 父亲请客时的工夫茶讲究 -> 他不理解有什么了不起
情绪：不解→委屈（强度6→7）
读者新获知：父亲对茶具有很深的执念
钩子：茶具到底有什么秘密
动静：静
对话密度：低

## 节5 | 回忆·高考填志愿
主事件：高考填志愿，父亲只说「你自己拿主意」
子事件：父亲沉默地看完志愿表 -> 只说了一句 -> 第二天母亲说「你爸半夜翻书」 -> 发现父亲买了大学地图
情绪：委屈→愧疚（强度7→8）
读者新获知：父亲不是不在意，是不知道怎么说
钩子：父亲买了一本大学地图册
动静：静
对话密度：零

## 节6 | 回忆·大学离家
主事件：陈潮生第一次离开潮汕去上大学
子事件：母亲煮了一大袋粿条非让他带 -> 父亲在车站转身就走 -> 母亲后来打电话说「你爸哭了」 -> 陈潮生愣住
情绪：冲击（强度8）
读者新获知：父亲哭了——这是铁打的男人第一次为儿子哭
钩子：他从来没见父亲哭过
动静：静
对话密度：低

## 节7 | 大学四年
主事件：大学期间和父亲之间「沉默」的通话模式
子事件：每次打电话母亲接 -> 父亲在旁边「嗯」一声就没了 -> 放假回家父亲还是不说话 -> 但母亲说「你爸说把虾留给你」
情绪：期待落空→隐约明白（强度7→8）
读者新获知：父亲的关心都在母亲转述的那些话里
钩子：一年暑假回去，发现父亲用他的房间当茶室
动静：静
对话密度：中

## 节8 | 结婚生娃
主事件：陈潮生结婚生子，父亲来城里看孙子
子事件：父亲第一次来深圳 -> 坐高铁都紧张 -> 抱着孙子手足无措 -> 住了两天就要回去，说「厝要有人看」 -> 临走塞了一个袋子，是攒的钱
情绪：心酸→愧疚（强度8→9）
读者新获知：父亲这辈子没离开过潮汕，但他为儿子来了城里
钩子：那个袋子里还有一包他爱吃的茶配
动静：动静
对话密度：中

## 节9 | 情感高潮·父亲住院
主事件：父亲摔伤住院，陈潮生在医院照顾
子事件：病房里的父亲瘦了一大圈 -> 迷迷糊糊醒来第一句话「茶盘带了没」 -> 陈潮生愣住 -> 回家拿茶具 -> 在父亲枕头下发现一本笔记本和一堆他的照片
情绪：泪崩（强度10）
读者新获知：父亲一辈子都在偷偷关注他，记事本上记满了他的一切
钩子：记事本上的字，歪歪扭扭的
动静：动→静
对话密度：中
伏笔/物件：工夫茶具第2现

## 节10 | 医院对话
主事件：陈潮生给父亲泡茶，父子第一次真正说话
子事件：父亲看着茶盘笑了 -> 陈潮生做工夫茶，手也抖了 -> 父亲说「潮生啊，爸不会说话」 -> 陈潮生说「我知道」 -> 两父子一杯茶喝了很久
情绪：释放→温暖（强度10→8）
读者新获知：父亲终于说出了那句「不会说话」，儿子终于说「我知道」
钩子：这杯茶喝了30年才喝懂
动静：静
对话密度：高

## 节11 | 给父亲泡茶
主事件：父亲出院后，陈潮生留在老厝陪了几天
子事件：每天给父亲泡工夫茶 -> 父亲教他「关公巡城要稳」 -> 他突然发现自己泡茶的动作和父亲一样 -> 问父亲那套茶具的来历
情绪：温暖→释然（强度7→6）
读者新获知：茶具是爷爷传下来的，三代人了
钩子：三代人的工夫茶
动静：静
对话密度：中
伏笔/物件：工夫茶具第3现（第一次真正意义使用）

## 节12 | 结尾
主事件：陈潮生回深圳后，自己也买了套工夫茶具
子事件：在家里泡工夫茶 -> 阿珍问这是什么 -> 他给儿子讲「关公巡城」 -> 儿子说「爷爷教过我」 -> 他手顿了一下
情绪：闭环余韵（强度6）
读者新获知：父亲早就在教孙子了，他以为父亲不爱说话，其实父亲用了一辈子在说
钩子（余韵）：潮汕男人的一生，话不多，茶很烫
动静：静
对话密度：中
