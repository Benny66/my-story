# 《五婿》小节大纲

**总字数目标**：30000字 / 25节（平均 ~1200字/节）
**节标记格式**：`###1.` 至 `###25.`
**对话格式**：`""`

###1. 五婿入门
**主事件**：大婚之日，五名赘婿依次入场，全城围观嘲笑
**子事件**：官差宣读圣旨(对话) -> 五人逐一进门，各显"废"态(冲突) -> 围观百姓议论纷纷，有人往门口扔烂菜叶(冲突) -> 沈青棠站在廊下，面无表情看着这一切(递进)
**情绪**：憋屈→冷
**读者新获知**：五个人有多"废"，女主有多镇定
**钩子**：最后一个人进门时，女主的瞳孔缩了一下
**伏笔/物件**：茶壶（第1现：她转身回屋，给自己斟了一杯茶）
**动静**：动（喧闹、嘲笑）
**对话密度**：中
**目标字数**：1200字

###2. 安置
**主事件**：入夜，她安置五个人的住处和生活
**子事件**：她给每人分配单独的房间并说了同样的话(对话) -> 陆深沉默了许久，只道了一声谢(发现) -> 季舟躲在门后发抖(发现) -> 她回房，发现谢浊醉倒在自己房门口(冲突)
**情绪**：冷→疑惑（第一次觉得不对劲）
**读者新获知**：女主对五人有基本的善意（不是恶毒主母）
**钩子**：她发现谢浊倒在自己门口，但她睡前锁了院门——他怎么进来的？
**伏笔/物件**：茶壶（她给每人也斟了一碗茶放在房内）
**动静**：静
**对话密度**：中
**目标字数**：1200字

###3. 十年前的半碗粥

**子事件**：梦中再现逃难路上的饥饿和绝望(回忆) -> 五个瘦骨嶙峋的少年蹲在路边，她犹豫后分了半碗粥(回忆) -> 醒来，天还未亮，她取出父亲留的玉扳指(发现) -> 对着扳指出神片刻，又藏了回去(反应)
**情绪**：苍凉→冷
**读者新获知**：女主有悲惨的过去，曾被贩卖过；她有过一枚重要的玉扳指
**钩子**：她把玉扳指贴在胸口的时候，窗外的树影动了一下——有人在屋顶？
**伏笔/物件**：玉扳指（第1现）、半碗粥（第1现）
**动静**：静
**对话密度**：零
**目标字数**：1000字

###4. 夜半剑影

###6. 醉话里的秘密

**钩子**：谢浊装睡时，嘴角微微勾了一下——他在笑，不是醉笑
**伏笔/物件**：（铺垫：谢浊的燕相背景，他掌握着六国时期的机密情报）

###8. 怯者的刀

###9. 五人齐聚

###22. 我的身份

**伏笔/物件**：玉扳指（第2现：身份的见证）

**目标字数**：1300字
