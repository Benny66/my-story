# 细纲 · 第3章：榕城面试奇遇记

- 字数目标：4000+
- 节奏：沙雕开场→紧张面试→情感收尾
- 时间线：返乡第3天

## 章首爆点
- 类型：沙雕名场面 + 熟人社会暴击
- 内容：女主穿着碎花裙去青潮面试，在中山路口被老妈牌友拦下来，阿姨拉着她的手说"长这么大了！你小时候我抱过你！" 聊了十分钟才脱身。冲到青潮门口，前台一抬头——强叔的女儿。女主内心：整个揭阳都是我妈的眼线。

## 核心事件
面试路上被熟人拦截 → 到青潮前台是强叔女儿 → 陆辞在开会（女主坐在沙发上观察他） → 面试正式开始 → 关键对话：陆辞问"甘心吗" → 面试通过，试用期一个月 → 回家路上 → 晚上读陆辞的文章 → 情感共鸣

## 本章爽点
- 沙雕开场：熟人社会暴击×2
- 面试交锋：陆辞的"甘心吗"是个好问题
- 陆辞形象完成：从背影→传闻→面对面→文章，四层递进

## 章尾钩子
- 类型：情感共鸣 + 新开始
- 内容：女主读完文章，发了条消息给阿敏："我过了。" 阿敏秒回："恭喜！明天下班出来喝糖水！" 第二天要去公司了。新的生活要开始了。
